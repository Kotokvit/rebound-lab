# Rebound Validation Pack
Adds CI scripts to *confirm or falsify* your architecture.
- `scripts/run_orbits.py`: integrate system, output `orbits.csv`, plots and `summary.json`.
- `scripts/tides_scan.py`: grid of tidal heating vs (a_s/R_H, e).

Run from GitHub → Actions → **Rebound validation** → *Run workflow*.
